API route descriptions and methods





















Route Method Description
Users
/users GET Fetch all user data
/user/:id GET Fetch a single user's data
/register POST Register a new user
/user/:id PATCH Edit a user's data
/user/:id DELETE Delete a user from the database
/login POST Logs a user in
Books
/books GET Fetch all books data
/book/:id GET Fetch data from a single book
/book POST Add a new book
/book/:id PATCH Edit a book's data
/book/:id DELETE Delete a book's record
Book Authors
/bookauthors GET Get all book authors' data
/bookauthor/:id GET Get data from a single book author
/bookauthor POST Add a book's author
/bookauthor/:id PATCH Edit a book author's details
/bookauthor/:id DELETE Remove a book author's record
Orders
/orders GET Fetch all orders' data
/order/:id GET Fetch a single order's data
/addOrder/:orderID/:userID/:bookID POST Add an order
/order/:orderID PATCH Edit an order
/order/:orderID DELETE Delete an order
